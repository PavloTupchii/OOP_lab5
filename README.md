## Опис реалізації

### `Coffee` Абстрактний базовий клас для кави.

* Поля: `name` (назва), `weight` (вага), `volume` (об'єм), `price` (ціна), `quality` (якість).
* Методи: геттери, обчислення співвідношення `getPriceToWeightRatio()`.

### `CoffeeBean` / `GroundCoffee` Конкретні класи-нащадки.

* Викликають базовий конструктор із зазначенням стану (у зернах / мелена).

### `InstantCoffee` Конкретний клас-нащадок для розчинної кави.

* Додатково має поле `packagingType` (тип упаковки: банка чи пакетик).

### `Van` Клас для роботи з фургоном:

* Поля: `maxVolume` (максимальний об'єм), `currentVolume` (поточний об'єм), `currentPrice` (загальна вартість), список `cargo` (вантаж).
* Додавання кави: `addCoffee()` з перевіркою вільного місця.
* Сортування: `sortByPriceToWeightRatio()` (за співвідношенням ціни до ваги).
* Пошук: `findByQualityRange()` (за діапазоном якості).

### `VanCapacityExceededException` Клас винятку.

* Обробляє ситуацію перевищення доступного об'єму фургона.

### `Main` Клас для демонстрації роботи:

* Створення об'єктів різних видів кави.
* Завантаження фургона та виведення початкового стану.
* Демонстрація сортування та пошуку.
* Демонстрація обробки виняткової ситуації (перевантаження).
